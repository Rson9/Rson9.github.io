1. 输出内容

1.1 Print

将内容输出到系统的标准输出

fmt.Print(a ...interface{})(n int ,err error) // 无换行
fmt.Printf(format string,a ...interface{})(n int,err error)  // 可格式化输出，无换行
fmt.Println(a ...interface{})(n int,err error) // 多一个换行
